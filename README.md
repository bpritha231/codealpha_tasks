# CodeAlpha UI/UX Design Task

## Project Title

Movie Ticket Booking Application

## Description

This project is a UI/UX design prototype created using Figma for a Movie Ticket Booking Application. The application allows users to browse movies, view movie details, select seats, book tickets, and make payments through an intuitive and user-friendly interface.

## Tools Used

* Figma

## Features

* User Authentication
* Browse Movies
* Movie Details Page
* Seat Selection
* Ticket Booking
* Secure Payment Interface
* Booking Confirmation
* User Profile and Booking History
